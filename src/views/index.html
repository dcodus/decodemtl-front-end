Overview

This UI toolkit is a highly-modular design system for rapid web page development of DecodeMTL. It contains different materials that can be assembled into more complex page layouts.

This guide contains real working examples, code snippets, documentation, and style guidelines.

Using The Styleguide

Use the menu on the left to navigate through the different elements of the styleguide.

  • The Typography and Components sections make up the most basic buidling blocks of the system.

  • The Modules sections use the components to create modulatr blocks which can be dropped within nay page layout.

  • The Footers and Navigation sections are unique site-wide modules.

  • The Pages section offers a complete view of the combined elements.
